## textarea

This renderer provides a textarea input.

### Properties

#### label

```json
{
  "label": "Bar",
  "model": "foo",
  "renderer": {
    "name": "textarea"
  }
}
```

#### placeholder

```json
{
  "model": "foo",
  "placeholder": "Bar",
  "renderer": {
    "name": "textarea"
  }
}
```

#### renderer.cols

```json
{
  "model": "foo",
  "renderer": {
    "cols": 4,
    "name": "textarea"
  }
}
```

#### renderer.rows

```json
{
  "model": "foo",
  "renderer": {
    "name": "textarea",
    "rows": 4
  }
}
```
